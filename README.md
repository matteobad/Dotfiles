# DOTFILES

## Getting Option 1
cd ~
git clone https://github.com/matteobad/dotfiles.git .dotfiles/ && cd .dotfiles

## Getting Option 2
cd ~
mkdir .dotfiles && curl -L https://github.com/matteobad/dotfiles/tarball/master | tar -xz -C .dotfiles/ --strip-components=1 && cd .dotfiles

## Install
source install.sh
